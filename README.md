# meven.rest

Предполагается, что данные присылаются через POST в виде одномерного массива
В method передается действие (add / update / remove / get)

Для add:
name - название
address - адрес

Для update:
name - название
address - адрес
id - id элемента

Для remove:
id - id элемента

Для get ничего не нужно
